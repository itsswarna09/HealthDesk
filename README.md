# Health Desk - Smart Digital Health Website

A responsive healthcare website for a smart digital health desk platform where users can book doctor appointments, manage medical records, request blood donors, and access emergency health support from one place.

## Features

- Professional landing page with healthcare service sections
- Doctor discovery and appointment booking form
- Specialty filters for appointments
- Medical records vault UI
- Blood donor request form and nearby donor list
- Emergency support actions and emergency contacts
- Testimonials, FAQ, and contact call to action
- Responsive layout for desktop and mobile

## Run Locally

Open `index.html` directly in a browser.

You can also run a local server:

```bash
python -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/index.html
```

## Deploy To GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `app.js`, and `README.md`.
3. Go to repository `Settings` > `Pages`.
4. Choose the main branch and root folder.
5. Save and open the generated GitHub Pages URL.

## Notes

This is a static frontend website. It uses sample data in `app.js` and external CDN assets for icons, fonts, and images.
